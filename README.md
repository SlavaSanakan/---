# Байесовская система динамической балансировки сложности (DDA)

Симуляция адаптивной системы балансировки сложности игры, использующей байесовский вывод и обучаемую матрицу параметров.

## Возможности

- **Байесовская оценка навыка** игрока в реальном времени
- **Адаптивная матрица параметров** (3x5) с обучением
- **5 регулируемых параметров сложности**:
  - Скорость спавна врагов
  - HP врагов
  - Скорость врагов
  - Шанс появления стрелков
  - Скорость пуль врагов

## Управление

| Клавиша | Действие |
|---------|----------|
| Q/A | Повысить/понизить точность игрока |
| W/S | Повысить/понизить силу убийств |
| E/D | Повысить/понизить выживаемость |
| SPACE | Пауза |
| R | Сброс системы |
| ESC | Выход |
